Open Source

The term open source refers to a software program or platform with source code that can be modified, enhanced or redistributed by anyone as its design is set to publically accessible.

Open source is originally associated with a software or IT development project with freely available code and permission to make edits, fix bugs, alterations, and enhancements.



It can be explained more basically that it is like a recipe of some meal.

If an open source recipe is public, it means that anyone can know the ingredients and instructions, and even upgrade the meal to improve its taste.


Examples of open source products:

  • WordPress (a content management system)
  • Open Office
  • The Internet browser Mozilla Firefox
  • Wikipedia
  • the GNU/Linux operating system
